umbrUI

Experimental Shadow DOM styling of <input> elements

What the Heck is Shadow DOM? So there! Basically this demo is about selecting those special pseudo elements and style them with CSS. And to make this Shadow-DOMing game even more fun, more rules: No images, no extra markup and no JavaScript (the browser already adds the functionality). If you're curious about the name umbrUI.. Umbra means shadow in latin.

Browser support: WebKit. Note that other browsers also allow Shadow DOM styling, but to keep things a little simpler, it's just WebKit for now. In case, here some screenshots. So, dear Copy&Paste kids.. this is just a demo and not meant to be used in production. As said above, it has no cross-browser support and didn't get tested at all.
